qiscus adalah perusahaan start-up yang berasal dari Singapura. qiscus adalah messaging to get the work done. Kami percaya bahwa komunikasi merupakan hal yang vital dalam produktifitas perusahaan manapun, dan berusaha membuat komunikasi dan kolaborasi menjadi efisien dan fun.

Kami mengundang developer-developer handal untuk bergabung dengan development dojo kami di Jogja. Saat ini kami memiliki peluang pekerjaan (full time/part time) di posisi berikut:

Benefits

  • Gaji yang kompetitif.
  • Jam kerja yang fleksibel.
  • Terbuka untuk kerja remote.
  • Bekerja dengan rekan kerja yang fun, pintar dan berpengalaman.
  • Peluang selebar-lebarnya untuk mengembangkan diri.
  • Pengalaman bekerja di sebuah startup international dengan tim dari Singapura, Malaysia dan Indonesia.

Mobile Developer (Android & iOS)

Requirements :

  • Menguasai android/ios development platform.
  • Penguasaan java atau objective-c/swift yang baik.
  • Berpengalaman dengan aplikasi yang terkoneksi dengan server web.
  • Berpengalaman dengan aplikasi yang sudah sampai tahap production stage.
  • Pengalaman di teknologi real time communication (websocket, webrtc, dll) akan menjadi nilai tambah.
  • Pengalaman di teknologi rendering opengl akan menjadi nilai tambah.
  • Pengalaman di bidang ui/ux sebuah aplikasi mobile akan menjadi nilai tambah.

DevOps

Requirements :

  • Menguasai server berbasis UNIX/Linux.
  • Penguasaan shell script yang baik.
  • Memiliki pengetahuan mengenai bahasa pemrograman seperti: python, ruby (rails), dan javascript (nodejs).
  • Memiliki pengetahuan mengenai basis data seperti: mysql, postgresql, dan redis.
  • Memiliki pengalaman dengan AWS akan menjadi nilai tambah.
  • Memiliki pengalaman dengan docker akan menjadi nilai tambah.
  • Memiliki pengalaman dengan automation tools seperti puppet atau chef akan menjadi nilai tambah.
  • Memiliki pengalaman dengan CI tools seperti jenkins, travis, atau codeship akan menjadi nilai tambah.
